# Resume PDF Placeholder

Add your resume PDF file as `resume.pdf` in this directory (public/).

The download link is configured in the Resume page at `/resume`.

To create a resume PDF, you can:
1. Export from Google Docs, Microsoft Word, or LaTeX
2. Use online resume builders
3. Convert from HTML using tools like wkhtmltopdf

Recommended: Keep the file size under 1MB for fast downloads.
